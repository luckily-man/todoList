# React 待办事项应用（本地存储版）

一个使用 **React + 本地存储 (localStorage)** 实现的简洁待办事项小应用，直接双击 `index.html` 即可在浏览器中使用。

## 功能特性

- 输入任务内容并添加到列表
- 每个任务前有**自定义圆形复选框**，点击即可标记完成
- 已完成任务会显示**删除线**效果
- 每个任务右侧有**删除按钮**，可单独删除
- 底部展示**全部 / 已完成 / 未完成数量**
- 响应式设计，在手机和桌面端都有良好体验
- 所有任务数据保存在浏览器 **localStorage** 中，刷新页面不会丢失

## 使用方式

1. 确保项目在目录：`d:\my\study`（或你自己的任意目录）
2. 直接用浏览器打开：
   - 方式一：在资源管理器中**双击 `index.html`**
   - 方式二：浏览器中使用 `Ctrl + O`，选择 `index.html`

> 说明：由于本项目使用的是 CDN 方式引入 React / ReactDOM / Babel，不需要本地安装依赖。

## 可选：使用本地静态服务预览

如果你安装了 Node.js，可以在命令行中这样预览：

```bash
cd d:\my\study
npx serve .
```

然后在浏览器访问终端输出的地址（通常是 `http://localhost:3000` 或类似端口）。
